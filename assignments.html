Dinosaurs. Yes, dinosaurs.

Check out our GitHub project's ruby101/assignments/week1/ folder. In it you will find dinosaur_before.rb.

Your task is to follow the directions in dinosaur_before.rb and fix all the issues in the code! The answers are in dinosaur_after.rb if you need help. Afterwards, try to refactor the code using attr_accessor (solution is in dinosaur_refactor.rb).

Zombie Twitter Part 1


What's worse than having people mad at you for not supplying them with a twitter application? Having people AND zombies mad at you for failing to do so. Your mission is to complete the code in assignments/week2/zombie_twitter.rb in order to be able to successfully run these commands:

application = Application.new 
application.users 
application.users.first.tweets 
application.tweets 
application.tweets.first.author 
application.topics

The code is currently commented with "# your code here" as helpful hints for where you should add your code.
