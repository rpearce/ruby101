Ruby 101

Class 1

In this class you will learn the fundamentals of the Ruby language and how to leverage its awesomeness in the e-Universe. We will also introduce your class project.

Topics Covered

  • Why Ruby?
  • Storing data in variables
  • if this then do that!
  • Save yourself time with loops

Students should

  • Understand the basics of Ruby
  • Understand the why behind why they are learning Ruby

Class 2

In this class we will discuss how to store and manipulate data via Arrays, Hashes, and Classes. We will also execute bits of code using Blocks and Methods.

Topics Covered

  • Arrays
  • Hashes
  • Blocks
  • Methods
  • Classes

Students should

  • Feel a little overwhelmed
  • Understand the different ways of storing and sorting data
  • Understand the different ways of organizing and executing code

Class 3

This class introduces the Ruby on Rails web framework. We will use it to build a small web application in class!

Topics Covered

  • What is Ruby on Rails?
  • RubyGems & libraries
  • Inheritance
  • MVC architecture

Students should

  • Feel a little overwhelmed
  • Understand how Rails utilizes basic Ruby to create something complex
  • Be able to start their own Rails project

Class 4

This class expands on the Ruby on Rails web framework and, in creating a new application, ties together all of the topics covered in the class.

Topics Covered

  • Creating a basic interface to create, view, update, and remove items
  • Routing and responding via Controllers
  • Using Models to store data in a database
  • Deploying applications via Heroku

Students should

  • See the different topics covered throughout the class come together to form a functional application
  • Have a decent grasp on the Ruby language
  • Understand the basic principles of Ruby on Rails
